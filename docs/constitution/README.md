## How to Use the Constitution

The Constitution is not a product requirements document. It does not describe features, interfaces, pricing or timelines.

It describes what CULO is for, what it stands for, and how it should behave — permanently. When a product decision, commercial arrangement, or community policy needs to be made, the Constitution is the reference for whether that decision is aligned with what CULO fundamentally is.

If a decision requires compromising what the Constitution says CULO is for, it is the wrong decision.
